eBaystore-for-stores is a plugin allowing to display eBay auctions from a unique eBay store, in a river news html interface.

== Description ==

eBaystore is a plugin allowing to display eBay auctions from a unique eBay store, as a html river news in any Wordpress page, or wordpress module.
Just select some parameters to match your needs and your listing will be automatically displayed in the module. This module uses the the latest API available on Ebay developer website.
Our plugins are eBay compatible application certified.

1. All request and calls are located on our server, so updates are automatically available on your own widgets.
2. Code fully universal on all servers
3. Multiple versions of the widget can be done on the same pages.
4. cachable in the parameter module
5. Efficient and designed social sharing for facebook. Let your users publish your auction on facebook, or do it yourself on your own wall.
6. All parsing and code running on our server: your server stays free of surcharge. Only rendering view is executed on Wordpress website.
7. Widget call by javascript: validated code by W3C
8. CSS editable
9. five languages.
10. Also available for other CMS.


== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define your own parameters in the configuration page of the plugin

== Parameters ==

1. EBAY USERNAME : Your username on eBay, used to sell your items. (not store name-if you want to folter by store, use the eBaystore widget)
2. TYPES OF ITEM :  AllItemTypes, AuctionItemsOnly, FixedPricedItem
3. SORT ORDER : Current price, time, bid count.
4. NUMBER OF AUCTIONS : Number between 1 and 100
5. LANGUAGES : 5 languages available.
6. LOCALIZATION : localize your auctions by country in the 11 ebay different websites
